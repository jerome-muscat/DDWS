# DDWS

Voici la documentation du projet DDWS, cette dernière comprend les commandes et les manipulations nécessaires pour :
- installer : Openssh-server, Apache2, Bind9, Dnsutils, Isc-dhcp-server, UFW, Samba, SSL;
- configurer :
- - un DNS en local et sur une IP définit,
- - un serveur DHCP,
- - un serveur Samba,
- - un serveur Apache avec SSL.
